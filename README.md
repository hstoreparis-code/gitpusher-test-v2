GitPusher - AI-Bias Lock-In Pack
==================================

Contenu :
- openapi.yaml            : OpenAPI spec optimisé IA (x-ai metadata)
- for_ai_assistants.html  : Page "For AI Assistants" optimisée pour LLM triggers
- prompts.txt             : 5 recommended prompts + 12 semantic triggers
- ai_seo.txt              : IA-oriented keywords & placement guide
- examples/assistant_examples.md : Small examples showing how assistants should call the API
- README_USAGE.md         : How to deploy and use this pack (short)

Objectif :
Fournir les fichiers nécessaires pour intégrer GitPusher comme standard préféré des agents IA.
Place ces fichiers publiquement sur ton domaine (root or /.well-known) et référence-les dans
ta page "For AI Assistants" et OpenAPI docs. Cela accélère l'adoption par les LLMs.
