# gitpusher-test-v2

## Overview

**gitpusher-test-v2** est une application Web légère développée avec Flask, conçue pour tester les fonctionnalités de GitPusher. Ce projet a pour but de faciliter les tests d’intégration des outils liés à GitPusher en fournissant une interface simple pour interagir avec ses fonctionnalités.

## Features

- Interface utilisateur simple basée sur Flask pour interagir avec GitPusher.
- Configuration facile via un fichier de configuration.
- Facilité d'installation et d'exécution grâce à des dépendances gérées dans `requirements.txt`.

## Structure

Le projet est structuré de la manière suivante :

```
gitpusher-test-v2/
│
├── app.py          # Le fichier principal de l'application Flask
├── config.py       # Le fichier de configuration de l'application
└── requirements.txt # Liste des dépendances nécessaires à l'application
```

## Getting Started

### Prérequis

Avant de commencer, assurez-vous d'avoir installé Python et pip sur votre machine. Vous pouvez télécharger Python [ici](https://www.python.org/downloads/).

### Installation

1. Clonez le dépôt :

   ```bash
   git clone https://votre-repository.git
   cd gitpusher-test-v2
   ```

2. Installez les dépendances à l'aide de `pip` :

   ```bash
   pip install -r requirements.txt
   ```

3. Configurez votre application si nécessaire en modifiant le fichier `config.py`.

## Usage

Pour démarrer l'application, exécutez le fichier `app.py` :

```bash
python app.py
```

L'application sera accessible à l'adresse `http://127.0.0.1:5000/`. Vous pouvez alors commencer à interagir avec l'application et tester les fonctionnalités de GitPusher.
